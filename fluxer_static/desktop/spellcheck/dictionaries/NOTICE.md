# Spellcheck dictionary licenses

The dictionaries in this directory are vendored from exact versions of the
`dictionary-*` npm packages generated by `wooorm/dictionaries`. Each dictionary
directory contains the upstream package license when the package ships one.

`dictionary-et@2.0.0` and `dictionary-eu@4.0.0` did not ship a standalone
`package/license` file in their npm tarballs, so their local `LICENSE` files
record the license statement from the upstream package README.
