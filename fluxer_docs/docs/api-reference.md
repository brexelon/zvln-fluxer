# API Reference

The Fluxer API is described by an OpenAPI specification. Use it to explore endpoints, generate client libraries, or import the API into tooling such as Postman, Insomnia, or a Swagger/Redoc viewer.

- **OpenAPI spec**: [`api.fluxer.app/openapi.json`](https://api.fluxer.app/openapi.json)
